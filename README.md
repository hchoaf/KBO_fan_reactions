# KBO_fan_reactions
Python toy project that creates a wordcloud from KBO fan's reactions

Used Python libraries:
- konlpy
- wordcloud
- beautifulsoup
- matplotlib
- urllib

NanumGothic font required in root directory

DC인사이드 KBO 구단별 갤러리에서 글 제목에서 빈도수 많은 단어로 단어구름을 생성 팬들의 반응 분석
